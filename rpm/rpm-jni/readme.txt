This project creates a RPM for an application that uses a native library.
Target os: linux

Getting the the nar file into a RPM was not trivial.

The generates nar file is OS specific, e.g. for 64 and 32 bit resp:
org.mahu.rpm.rpm-jni-0.1.0-SNAPSHOT-amd64-Linux-gpp-jni.nar
org.mahu.rpm.rpm-jni-0.1.0-SNAPSHOT-i386-Linux-gpp-jni.nar
